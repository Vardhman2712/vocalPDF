PDF Reader with Text Highlighting:

This project is a dynamic website that allows users to upload a PDF file and listen to the content being read aloud. As the content is being read, the corresponding text on the webpage is highlighted in real-time, providing an interactive experience for the user. The website is built using HTML, JavaScript, Python (Flask), and pyttsx3 for text-to-speech functionality.

Features:
    Upload a PDF file to the website.
    Listen to the content of the PDF read aloud.
    The text being read is highlighted in real-time on the webpage.

Tech Stack:
    Frontend: HTML, CSS, JavaScript
    Backend: Python, Flask
    Text-to-Speech: pyttsx3 (Python library)
    PDF Parsing: PyPDF2 or PyMuPDF (Python libraries)
